# Snake Game üéÆ

This is a classic Snake game implementation built as a practice project to learn game logic, keyboard input handling, and basic rendering. The goal is to control the snake, eat food to grow longer, and avoid running into the walls or your own tail.

## Features

- Classic Snake gameplay with continuous movement.
- Food spawns at random positions on the board.
- Score increases as the snake eats food.
- Game over when the snake collides with itself or the boundary.
- Simple, readable code structure ideal for beginners to study.
